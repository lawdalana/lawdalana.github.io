## 🔓 Heretic vs Abliterated — ความต่าง

| | Heretic | Abliterated |
|---|---------|-------------|
| **วิธี** | Fine-tune ด้วย uncensored data | ลบ refusal direction จาก weights |
| **เทรนเพิ่ม?** | ✅ มี | ❌ ไม่มี |
| **เปลี่ยน personality?** | ✅ เปลี่ยน | ❌ เหมือนเดิม |
| **Uncensor สมบูรณ์?** | ✅ แน่นอน | ⚠️ อาจไม่ 100% |
| **ความรู้เดิม?** | ⚠️ อาจหายบ้าง | ✅ รักษาได้ |
| **Benchmark?** | ⚠️ อาจตก | ✅ คงที่ |

> **Heretic** = Fine-tune ด้วย uncensored dataset → เปลี่ยน behavior ทั้ง model → uncensor สมบูรณ์แต่อาจเสียความรู้
> **Abliterated** = หา "refusal direction" แล้วตัดออกจาก weights → ไม่เทรนเพิ่ม → รักษาความรู้เดิมแต่อาจ uncensor ไม่ 100%
